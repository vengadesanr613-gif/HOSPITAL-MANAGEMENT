# HOSPITAL-MANAGEMENT
The Hospital Management System is a web-based application designed to manage hospital activities efficiently. It helps in handling patient records, doctor details, appointments, billing, and staff information through a centralized system.

## 📌 Project Description
The Hospital Management System helps manage hospital activities such as
patient registration, doctor details, appointments, and basic hospital information.
This project is suitable for students learning web programming.

## 🛠️ Technologies Used
- HTML

## ✨ Features
- Hospital homepage
- Patient registration form
- Simple and user-friendly interface
